Download & installation

Swift modeller

Download SWIFT MODELLER v2.0 by clicking here.

If you wish to obtain SWIFT MODELLER v1.0 please mail to shankaracharya@bitmesra.ac.in. We will mail you the link for downloading.

 

Prerequisites:

The following must be pre-installed in the system

· MODELLER

· Python (version should be compatible to MODELLER)

· Numpy

· Matplotlib (versions of numpy and matplotlib should be compatible to Python)

· Java Runtime Environment

 

Tested Operating Systems:

· Microsoft Windows XP, Vista and 7

· Fedora, Red Hat Linux

· Ubuntu

· MacOSX

 

Note:

· Microsoft Windows Vista or 7 users must install MODELLER in a partitioned drive other than C:\ to avoid administrator rights related hassles.

· Linux is available in various distributions. If you use test SWIFT MODELLER v2.0 on any other OS (for which MODELLER distribution exists) other than those mentioned, then kindly notify us so that we can add it to our list of OS.
